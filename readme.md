## Sobre o Sitema Financeiro

Sistema feito em Laravel 5.8 com AdminLte, e Mysql, pronto para uso. Neste sistema o foco é controlar o que deve ser pagou ou recebido. 

## O que podemos fazer no financeiro

- [X] Cadastrar, Editar e Excluir Centros de custos;
- [X] Cadastrar, Editar e Excluir Contas;
- [X] Cadastrar, Editar e Excluir Entradas;
- [X] Cadastrar, Editar e Excluir Saídas;
- [X] Cadastrar  Saída Rápida;
- [X] Cadastrar, Editar e Excluir Categorias de Entradas ou Saídas;
- [X] Cadastrar, Editar e Excluir Usuários/Funcionários;
- [X] Cadastrar, Editar e Excluir Fornecedores;
- [X] Cadastrar, Editar e Excluir Clientes;
- [X] Efetuar Saques, Depósitos e Transferências entre contas;
- [X] Histórico de Transações;
- [X] Relatório de Saídas por contas, Fornecedores e categorias;
- [ ] Relatório de Entradas;
- [ ] Salvar Comprovantes de Pagamento;
- [X] Relatório de Saídas;
